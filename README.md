# Criptografia com Next.js
==========================

Este é um projeto de criptografia utilizando Next.js e Tailwind CSS.

## Funcionalidades

* Criptografia de texto utilizando algoritmo de criptografia
* Autenticação de usuários
* Interface de usuário responsiva utilizando Tailwind CSS

## Tecnologias utilizadas

* Next.js
* Tailwind CSS
* JavaScript
* HTML
* CSS

## Instalação

1. Clone o repositório: `git clone https://github.com/usuario/projeto.git`
2. Instale as dependências: `npm install`
3. Inicie o servidor de desenvolvimento: `npm run dev`

## Uso

1. Acesse o projeto em `http://localhost:3000`
2. Insira o texto a ser criptografado no campo de texto
3. Clique no botão "Criptografar" para criptografar o texto
4. Insira a senha para autenticar o usuário
